# WhatsApp Web

  <div className="project-content">

    <div className="project-description">
      <div className="description-text">
        WhatsApp Web è una replica responsive dell'interfaccia web di WhatsApp, sviluppata con Vue.js, JavaScript, HTML e CSS. Il progetto simula un'applicazione di messaggistica, consentendo agli utenti di visualizzare una lista di contatti, selezionare una conversazione e inviare o ricevere messaggi.
      </div>

      <div className="description-text">
        Le principali caratteristiche includono:
      </div>

      <ul> <li><strong>Gestione dinamica dei contatti:</strong> Utilizzando la direttiva <code>v-for</code> di Vue.js, l'applicazione genera dinamicamente la lista dei contatti, permettendo una facile scalabilità e manutenzione del codice.</li> <li><strong>Barra di ricerca reattiva:</strong> La barra di ricerca permette di filtrare i contatti in tempo reale mentre l'utente digita, offrendo un'esperienza fluida e interattiva grazie al binding reattivo di Vue. </li> <li><strong>Simulazione di messaggistica:</strong> L'app consente l'invio e la ricezione di messaggi, replicando le funzionalità base di un client di messaggistica come WhatsApp.</li> </ul>
    </div>
